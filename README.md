Emarsys SDK based on [Emarsys API v3](https://dev.emarsys.com/docs/core-api-reference).

## Installation

You can install the package via composer:

```bash
composer require hobbii/emarsys-sdk-php
```

## Usage

```php
$client = new Hobbii\Emarsys\Client();
```

## Testing

```bash
composer test
```
